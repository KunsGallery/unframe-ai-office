# Pixel Agents Assets

Source: https://github.com/pablodelucca/pixel-agents  
Author: Pablo De Lucca  
License: MIT, where applicable in the source repository.

Usage in UNFRAME AI Office:
- Character sprite assets for visualizing AI agents.
- No VS Code extension logic is used.
- No Claude Code JSONL watcher is used.
- No terminal watcher is used.

Excluded:
- VS Code extension source logic
- Claude Code session watcher
- external paid or unclear-license tilesets and furniture assets

- Furniture, floors, and wall assets were intentionally not copied in this step to avoid including any asset category whose downstream licensing context could be ambiguous for this web app.
